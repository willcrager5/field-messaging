# Session trace template

Copy this template to `Brand/Content/_sessions/YYYY-MM-DD-HHMM.md` at the end of every substantive session. Keep it terse — half a page at most. This is a diagnostic log, not a report.

---

## Notes on writing the trace

- End of session, not middle. Trace captures the whole session at once.
- Terse beats complete. A half-page trace read at 6pm tomorrow is more useful than a two-page trace no one opens.
- Every trace file is append-only from the moment it's written. If the session continues later, start a new trace file for the second session.
- File naming: `YYYY-MM-DD-HHMM.md` (24-hour local time of session start).
