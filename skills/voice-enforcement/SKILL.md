# Voice Enforcement — Field Brand Standards

Load this skill at the start of any session where you will write or review Field-branded content. Both enforcement guides below are active constraints for the session — not checklists to run at the end.

---

## How to apply these rules

**When writing:** Both guides are live constraints during drafting. Do not produce output that violates them. The test on everything you write: could Bill have written this?

**When reviewing:** Run anti-AI tells first (Category 4 openers/closers, then Category 1 structural, then Category 2 reasoning, then Category 3 sentence-level). Then run banned phrases. Flag every hit by name. Rewrite from the claim outward — don't patch in place.

---

## Part A — Anti-AI Tells

AI writing can pass every vocabulary check and still read like AI. These are the structural, reasoning, and rhythmic patterns that give it away.

### The read-aloud test

Before any checklist: read the content aloud. AI writing has a specific sound — even pace throughout, every sentence grammatically complete, paragraphs that begin, develop, and close. No acceleration, no compression, no fragments.

Bill's writing sounds different. Short sentences after long ones. Fragments used as emphasis. Paragraphs that lean into the next without closing. Key words repeated deliberately for rhythm, not varied for the sake of variety.

If it sounds smooth all the way through, it needs work.

---

### Category 1 — Structural Tells

**1.1 Announces before doing**
Flag: "Let me walk through how this works." / "Here's how to think about it:" / "Here's the thing:" / "This brings us to..." / "Let me explain what I mean."
Fix: Delete the announcement. Start the thing.

**1.2 Restates what it just said**
Flag: "In short, [restatement]." / "To put it simply," / "In other words," / "At the end of the day,"
Fix: Delete the closing restatement. The paragraph already made the point.

**1.3 Narrates its own structure**
Flag: "Now that we've covered X, let's turn to Y." / "As mentioned earlier..." / "This is where Field comes in." / "This is why [X] matters."
Fix: Remove the narration. Move to the next point.

**1.4 Argues with itself**
Flag: "One might argue X. However, Y." / "While X is true, it's also important to consider Y." / "Some might say X. But the reality is Y."
Fix: Make the claim. If an objection is real, address it on its own terms.

**1.5 Explains implications after stating them**
Flag: "What this means for advisors is..." / "The implication here is..." / "This is significant because..." / "This matters because..."
Fix: Let the point land. If the implication isn't obvious, rewrite the point.

**1.6 Announces complexity before handling it**
Flag: "This is a nuanced question." / "The answer is more complicated than it might seem." / "There's a lot to unpack here."
Fix: Handle the complexity. Don't announce it.

### Category 2 — Reasoning Tells

**2.1 False balance**
Flag: "While largely X, there are cases where Y." / "In many ways, Field is..." / "For the most part, X." / "In a sense, X."
Fix: Make the claim. If there's a real exception, name it specifically.

**2.2 Qualifier cluster at the end**
Flag: "...particularly in today's environment." / "...especially as the industry continues to evolve." / "...in the current market landscape."
Fix: Cut the tail. The claim stands or it doesn't.

**2.3 Soft hedge before a strong claim**
Flag: "In many ways, Field represents..." / "It could be argued that..." / "There's a case to be made that..." / "It's fair to say that..."
Fix: Remove the hedge. State the claim.

**2.4 Agency attributed to abstractions**
Flag: "The data tells us..." / "The market suggests..." / "The numbers reveal..." / "The trends indicate..."
Fix: Name who the data tells, or state the implication directly.

**2.5 The false anecdote**
Flag: "Imagine you're an advisor sitting at your desk..." / "Picture a firm principal looking at three different dashboards..."
Fix: Describe the actual condition. No staged scenes.

**2.6 Perfect logical flow**
Every paragraph flows seamlessly into the next. Real writing has detours. If everything flows too cleanly, find where a real writer would have stopped, reconsidered, or compressed.

### Category 3 — Sentence-Level Tells

**3.1 Transition crutches**
Flag: "It's worth noting that..." / "It bears mentioning that..." / "It's important to note..." / "Moreover," / "Furthermore," / "Additionally," / "That said," / "With that in mind,"
Fix: Delete the transition. Connect ideas directly, or let them sit as separate sentences.

**3.2 Gerund openers**
Flag: "Building on this..." / "Recognizing that..." / "Bringing together..." / "Leveraging the foundation..." / "Understanding the challenge..."
Fix: Rewrite to start with the subject or the action.

**3.3 The "not just X, but Y" escalation**
Flag: "Field is not just a data layer. It's a transformation." / "This isn't just about integration. It's about intelligence."
Fix: Make one clear claim. The escalation performs drama without adding content.

**3.4 Adverb hedging**
Flag: "quite" / "rather" / "fairly" / "somewhat" / "largely" / "mostly" / "essentially" / "fundamentally" / "inherently" / "genuinely" / "particularly" used as vague emphasis
Fix: Cut the adverb. If the modification is real, make it specific.

**3.5 Compound emotional adjectives**
Flag: "thoughtful and nuanced" / "bold and confident" / "clear and compelling" / "simple yet powerful"
Fix: Pick one or cut both.

**3.6 No fragments**
If every sentence is grammatically complete, it's probably AI. Real writing uses fragments deliberately for rhythm and emphasis. Find the moments that deserve a fragment. Use them.

**3.7 Perfect parallelism**
AI makes every element in a list the same grammatical structure and length. Real writing varies shape and length — the variation is the rhythm. Read the list aloud. If every element sounds the same length, break one of them.

**3.8 Vocabulary variation to avoid repetition**
AI cycles through synonyms to avoid repeating words. Bill repeats key words deliberately for emphasis: "The picture is fragmented. The picture should be whole." — not "That unified view should already exist." If a word appears multiple times and it's the right word, keep it.

---

### Category 4 — Opener/Closer Tells

**4.1 Epochal openers**
Flag: "In today's rapidly evolving landscape..." / "As wealth management continues to change..." / "In an era defined by..." / "Amid the growing complexity of..."
Fix: Start with the specific condition.

**4.2 "At its core" openers**
Flag: "At its core, Field is..." — AI windup before a definition.
Fix: State what it is. Drop the windup.

**4.3 Aspirational closers**
Flag: "...and that's the future we're building toward." / "...because that's what this industry deserves." / "...that's what drives everything we do."
Fix: End on the claim, the fact, or the action.

**4.4 The emotional tag**
Flag: "And that's what makes Field different — not just for firms, but for the advisors who run them." / "That's not just good for the firm. It's good for the advisor."
Fix: The argument already carries the emotion. Remove the tag.

**4.5 The "this is where X comes in" frame**
Flag: "This is where Field comes in." — sets up a reveal instead of stating the thing.
Fix: State what Field does. Skip the frame.

## Part B — Banned Phrases

Remove or replace every item below. No exceptions. Flag every hit by name and propose a specific replacement.

### Hard-ban expression patterns

**A. The "In a World Where" Drama**
Flag: "In a world where..." / "In today's fast-paced world..." / "In an era where..."
Fix: Today, [specific change]. So [practical implication].

**B. The "Most People vs. Few Who" Split**
Flag: "Most people X. But the ones who win Y..."
Fix: Among [group], [specific stat]. What works: [one tactic].

**C. The "Stop X. Start Y." Switch**
Flag: "Stop X. Start Y." / "Quit X. Do Y instead."
Fix: If you're doing X, switch to Y when [condition], because [reason].

**D. The "Not This. Not That. But This." Triple**
Flag: Three-part "It's not / It's not / It's" construction.
Fix: Make one clear claim.

**E. The "If You're Not Doing X, You're Already Behind" FOMO**
Flag: "If you're not X yet..." / "You're already behind if you haven't..."
Fix: State the trade-off plainly.

**F. The "The Real Work Is…" Reveal**
Flag: "The real X isn't Y. It's Z." / "Most people focus on X. The real issue is Y."
Fix: Spell out the actual work with steps.

**G. The "You Don't Need More X. You Need Y." Minimalist Smack**
Flag: "You don't need more X. You need Y."
Fix: State thresholds or a concrete checklist.

**H. The "It's Never Been Easier / Harder" Paradox**
Flag: "Never been easier to X. Never been harder to Y."
Fix: Name the specific trade-off.

**I. The "Here's the Truth / Nobody Tells You" Fake Reveal**
Flag: "Here's the truth:" / "What nobody tells you" / "The dirty secret is"
Fix: State the fact directly with detail.

**J. The "It Is Not This. It Is That." Binary**
Flag: "It is not X. It is Y." used as a standalone rhetorical move.
Fix: Make the one true claim directly.

**K. Em Dashes as Rhetorical Decoration**
Flag: Em dashes used to create a pause effect or introduce a punchy reveal — "X, Y, Z — [punchy conclusion]."
Fix: A period. Em dashes are acceptable for a true parenthetical aside only.

---

### Style rules

- Varied sentence structure and length. Short sentences after long ones.
- Everyday words. If a simpler word exists, use it.
- Direct statements over moralizing. State the thing.
- No hype tone. No exclamation marks in body copy. Claims land harder when stated plainly.
- Don't invent. No new stories, numbers, or promises that weren't in the source.
- Same length. Don't expand or compress significantly when editing.
